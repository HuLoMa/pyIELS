
# List of examples in the toolbox

## Electron Wigner functions

- demo_wigner_01: Compute the classical Wigner function after propagation
- demo_wigner_02: Compute the quantum Wigner function for a Fock state with $\overline{n}=1$
- demo_wigner_03: Explore the semi-classical limit of PINEM. The code compares PINEM with on a coherent state with $\overline{n}=25$ with the semi-classical result
